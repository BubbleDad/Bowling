# Meowmoon Bowling v1.5.7 Cardinal, Hanbok, and Ferry Animations Report

## Purpose

This release adds four new approved special pin animations from `meowmoon_codex_bundle_v2.zip`.

## Registry entries and durations

- `cardinalglide` — Cardinal Glide — 4200 ms
- `cardinalflyoff` — Cardinal Flies Off Screen — 4300 ms
- `hanbokribbontwirl` — Hanbok Ribbon Twirl — 3400 ms
- `merseyferry` — Mersey Ferry Glide — 3600 ms

All four use the existing `staticTimed` special-pin lifecycle and render as overlays.

## Approved sources

- Cardinal Glide v2: `cardinal-glide-review-v2.html`
- Cardinal Flies Off Screen v1: `cardinal-flies-off-screen-review-v1.html`
- Hanbok Ribbon Twirl v2: `hanbok-ribbon-twirl-review-v2.html`
- Mersey Ferry Glide v1: `mersey-ferry-glide-review-v1.html`

Mersey Ferry v1 was used. No Mersey Ferry v2 source was present or integrated.

## Extracted reusable functions

- `cardinalv2Draw` and prefixed math/star helpers.
- `cardinalflyoffv1Draw` and prefixed math/star helpers.
- `hanbokv2Draw` and prefixed math/star helpers.
- `merseyferryv1Draw`, `merseyferryv1DrawCloud`, and prefixed math/star helpers.

Standalone canvas setup, controls, resize handling, backgrounds, review loops, and review-only DOM code were not copied.

## Integration adjustments

- Namespaced generic helpers under each approved source prefix to avoid conflicts.
- Converted Cardinal Glide's absolute review coordinates to equivalent anchor-relative travel while preserving its double size, long flight, branch landing, and finishing star.
- Converted Cardinal Fly Off to a full-canvas anchor-relative stage so the doubled bird visibly exits right and never lands.
- Preserved Hanbok v2's double scale and clamped its anchor horizontally to avoid severe edge clipping.
- Converted Mersey Ferry's page-sized water and travel coordinates into an anchor-relative local water stage while preserving v1 scale, ferry proportions, and three rippling waves.
- Added drawing-only wrappers that supply the hit-pin anchor, scale, elapsed time, and registered duration.

## Intentionally not changed

- No scoring, gameplay, pin-count, startup-overlay, phase-state, lifecycle-helper, ball-special, turn-completion, or unrelated visual-design changes.
- No random values, frame counters, external assets, libraries, network calls, or asynchronous loading were added.
- No service-worker behavior changed beyond the required versioned app and cache references.
