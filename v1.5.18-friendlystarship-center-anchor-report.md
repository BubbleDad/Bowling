# v1.5.18 Friendly Starship center-anchor report

## Change summary

Friendly Starship Glide now renders from the logical screen center instead of the triggering pin position. This prevents edge-triggered placements from clipping the starship stage, planet, or sparkle trail.

## Animation details

- Registry key: `friendlystarshipglide`
- Draw wrapper: `friendlystarshipglidev1DrawPin`
- Duration: `3600` ms
- Placement fix location: draw wrapper
- Coordinate system: game logical `view.w` / `view.h`
- Center anchor: `view.w * 0.5`, `view.h * 0.5`
- Scale adjustment: none

## Behavior preserved

- Trigger logic remains pin-based.
- Only the starship visual placement ignores the trigger pin coordinates.
- Other special animations retain their normal pin-relative placement.
- No external assets, network calls, or third-party libraries were added.
