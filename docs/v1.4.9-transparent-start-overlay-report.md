# Meowmoon Bowling v1.4.9 Transparent Start Overlay

## Purpose

This release keeps the successful v1.4.8 HTML start overlay mechanism, but makes it visually invisible.

## What changed

- Removed the visible overlay card.
- Removed the overlay text.
- Changed the overlay background to transparent.
- Kept the overlay as a full-screen tap target so the first tap still reliably starts the game.

## What did not change

- No animation changes.
- No gameplay rule changes.
- No pin count changes.
- No special animation frequency changes.
- No change to the startup repair logic except the overlay appearance.

## Version/cache

This release uses:

- app file: `app-v1.5.0.js`
- cache name: `meowmoon-bowling-v1-5-0-wording-cache`
