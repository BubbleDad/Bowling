# Meowmoon Bowling v1.4.4 Phase-State Cleanup

## Purpose

This is cleanup implementation 3. It centralizes phase transitions and ball-loading state so the game is less likely to enter inconsistent states.

## Main change

Added phase helpers:

- `setGamePhase(nextPhase, reason)`
- `clearBallAndPath()`
- `enterPlayingPhase(message, forceHitNext, reason)`
- `enterRollingPhase(ball, path, reason)`
- `enterResolvingPhase(resolveUntil, message, reason)`
- `enterRewardPhase(reason)`
- `enterPausedPhase(reason)`
- `resumePhaseAfterPause()`
- `validatePhaseState(current)`

## What this protects against

- Rolling phase with no ball.
- Rolling phase with an invalid ball path.
- Resolving phase with a stale ball.
- Playing or reward phase with a stale ball.
- Playing phase when no upright pins remain.

## What did not change

- No new animations.
- No animation art changes.
- No pin count changes.
- No new gameplay rules.
- No intended behavior change during normal play.
