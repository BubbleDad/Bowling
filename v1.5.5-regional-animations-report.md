# Meowmoon Bowling v1.5.5 Regional Animations Report

## Purpose

This release adds four approved special pin animations.

## Registry entries

- `millyardgearv2` — Millyard Gear Machine — 3600 ms
- `lightrailv2` — Light Rail Glide — 3800 ms
- `raptorwingv2` — Raptor Wing Glide — 3900 ms
- `raptorleftv1` — Raptor Up Left Exit — 3900 ms

All four use the existing `staticTimed` special-pin lifecycle and render as overlays.

## Extracted functions

- Millyard: `millyardgearv2DrawMachine`, `millyardgearv2DrawGear`, and prefixed math helpers.
- Light rail: `lightrailv2DrawScene`, `lightrailv2DrawCar`, and prefixed math helpers.
- Raptor glide: `raptorwingv2DrawBird` and prefixed math helpers.
- Raptor left exit: `raptorleftv1DrawBird` and prefixed math helpers.

Standalone canvas setup, controls, review rendering, backgrounds, and responsive review-page layout were not copied.

## Integration adjustments

- Renamed generic review math helpers under each approved animation prefix to prevent collisions.
- Replaced the light rail's review-page width dependency with a deterministic stage width derived from its supplied anchor and scale; the car still travels fully off screen.
- Replaced the left-exit raptor's absolute review-page destination with deterministic anchor-relative departure distances that guarantee an upper-left exit.
- Added drawing-only wrappers that supply each hit pin's anchor, game scale, elapsed time, and registered duration.

## Intentionally not changed

- No gameplay, pin-count, startup-overlay, phase-state, lifecycle-helper, ball-special, or unrelated visual-design changes.
- No service-worker behavior changes beyond the required versioned app and cache references.
