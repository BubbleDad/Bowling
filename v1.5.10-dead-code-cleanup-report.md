# v1.5.10 Dead-Code Cleanup

This focused release removes 49 named function declarations that had no references outside their own declarations.

## Removed code

- Obsolete pin helpers: `restorePin`, `activePinCount`.
- Unused generic drawing helper: `drawSparkleRing`.
- Forty-six unused, animation-prefixed interpolation, easing, star, and cloud helpers associated with the approved special-animation implementations.

The registered animation draw functions, registry entries, durations, pin lifecycle, scoring, turn completion, level progression, startup overlay, and visual output are unchanged.

## Validation

- JavaScript syntax checked with `node --check`.
- All 49 removed identifiers confirmed absent from the new app file.
- Startup, normal rolling, representative special-animation playback and completion, and level progression regression-tested in the browser with no console errors.
