# Meowmoon Bowling v1.5.8 Eight-Animation Bundle Report

## Purpose

This release replaces Cardinal Glide v2 with approved v3 and adds seven new approved special pin animations from `meowmoon_codex_bundle_v3.zip`.

## Registry entries and durations

- `cardinalglide` — Cardinal Glide v3 — 4200 ms
- `tropicalbirdhop` — Tropical Bird Hop v2 — 3800 ms
- `rainbowbridgepop` — Rainbow Bridge Pop v2 — 3200 ms
- `mosaictileshimmer` — Mosaic Tile Shimmer v2 — 3400 ms
- `columncircledance` — Column Circle Dance v2 — 3400 ms
- `orangewhitegreenribbon` — Orange White Green Ribbon v1 — 3400 ms
- `streetcarbellroll` — Streetcar Bell Roll v1 — 3500 ms
- `campwagonroll` — Camp Wagon Roll v1 — 3500 ms

All eight use the existing `staticTimed` special-pin lifecycle and render as overlays.

- `cardinal-glide-review-v3.html` → `cardinalv3Draw`
- `tropical-bird-hop-review-v2.html` → `tropicalbirdv2Draw`

- `camp-wagon-roll-review-v1.html` → `campwagonv1Draw`

Only required prefixed drawing helpers were retained. Review controls, canvas setup, resize handling, background clearing, animation loops, DOM code, and unrelated dispatch branches were excluded.

## Integration adjustments

- Removed the v1.5.7 Cardinal Glide v2 functions and wrapper. The `cardinalglide` registry entry now calls only `cardinalv3DrawPin`; v3 contains no brown stick.
- Converted Cardinal v3's absolute review coordinates to equivalent anchor-relative travel while preserving double size, long flight, rest position, and finishing star.
- Localized Tropical Bird's ground strip and converted its takeoff to a full-canvas right exit while preserving exactly two hops.
- Removed review-only sky backgrounds from Rainbow Bridge, Mosaic, Columns, and Cardinal; their subject artwork is unchanged.
- Preserved Rainbow Bridge v2 without any sparkle calls.
- Preserved Mosaic's explicit 6-row by 7-column layout.
- Preserved Column Dance's five doubled-size columns and blue–pink–white–pink–blue order.
- Extracted only the requested ribbon and streetcar branches from their multi-animation review files, namespaced their helpers, and discarded all unrelated branches.
- Converted Streetcar and Camp Wagon page-width travel into deterministic full-canvas anchor-relative stages so both exit cleanly.
- Added drawing-only wrappers that supply pin placement, scale, elapsed time, and registered duration.

## Intentionally not changed

- No scoring, gameplay, pin count, startup overlay, phase state, lifecycle helper, ball special, turn completion, or unrelated visual design changed.
- No random values, frame counters, external assets, libraries, network calls, or asynchronous loading were added.
- No service-worker behavior changed beyond the required versioned app and cache references.
