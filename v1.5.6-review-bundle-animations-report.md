# Meowmoon Bowling v1.5.6 Review Bundle Animations Report

## Purpose

This release adds five approved special pin animations from `meowmoon_codex_bundle_v1.zip`.

## Registry entries and durations

- `tugboatbob` — Tugboat Bob — 3600 ms
- `jellybeanrowcolor` — Jellybean Row Color Pop — 3400 ms
- `tulipbloomwave` — Tulip Bloom Wave — 3600 ms
- `ribbondance` — Ribbon Dance — 3600 ms
- `metrotrain` — Metro Train — 3800 ms

All five use the existing `staticTimed` special-pin lifecycle and render as overlays.

## Extracted reusable functions

- Tugboat: `tugboatv1Draw` plus prefixed clamp, lerp, easing, and star helpers.
- Jellybean row: `jellybeanv2Draw`, `jellybeanv2DrawHouse`, and prefixed helpers.
- Tulips: `tulipsv3Draw`, `tulipsv3DrawTulip`, and prefixed helpers.
- Ribbon: `ribbondancev3Draw` plus prefixed helpers.
- Metro: `metrotrainv2Draw` plus prefixed helpers.

Standalone canvas setup, controls, resize handling, backgrounds, and review animation loops were not copied.

## Integration adjustments

- Namespaced every generic source helper under its approved animation prefix to avoid collisions.
- Preserved the review files' doubled v2 jellybean scale, large v3 tulip scale, and doubled-again v3 ribbon scale.
- Clamped wide stationary-scene anchors horizontally so hit pins near an edge do not create unintended clipping or giant offsets.
- Replaced the metro review page-width dependency with an anchor-relative full-canvas stage; the metro remains deterministic and exits completely to the right.
- Added drawing-only wrappers that supply the hit-pin anchor, game scale, elapsed time, and registered duration.

## Intentionally not changed

- No gameplay, pin-count, startup-overlay, phase-state, lifecycle-helper, ball-special, scoring, or unrelated visual-design changes.
- No network calls, libraries, or external assets were added.
- No service-worker behavior changed beyond the required versioned app and cache references.
