# v1.5.13 Legacy Branch Cleanup Report

## Scope

Performed the follow-up maintenance pass recommended after v1.5.12.

This is a cleanup-only release. It removes unreachable branches for legacy special pin types that are no longer registered in `SPECIAL_ANIMATION_REGISTRY`.

## Removed legacy code paths

- `treasure`
- `fish`
- `penguin`
- `wheelchairsprint`
- `volleyballspike`
- `skijump`
- `gymnasticsflip`

## Evidence

- None of the removed special pin type strings are present in the current registry.
- `PIN_SPECIAL_TYPES` is built from `Object.keys(SPECIAL_ANIMATION_REGISTRY)`, so regular pin assignment cannot select those removed types.
- The removed code was limited to launch/update/audio branches for those unreachable type strings.

## Intentionally retained

- `makeTreasureBurst()` remains because `makeBusBurst()` still calls it.
- `makeSportBurst()` remains because registered sports animations still call it.
- `meteor` branches remain because they are tied to ball-special handling/rendering, not the special pin registry cleanup target.

## Validation

- `node --check game/app-v1.5.13.js` passed.
- `node --check game/sw.js` passed.
- Declaration-only audit found 0 function declarations and 0 top-level declarations left unused after this cleanup.
- Registry count remains 71.

Manual browser regression should focus on ordinary startup, one normal roll, and a registered sports special animation to confirm the retained sport burst path.
